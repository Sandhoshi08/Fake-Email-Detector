# 📧 Fake Email Detector

A Machine Learning-based web application that detects whether an email is **Fake (Phishing)** or **Legitimate**. The project uses Natural Language Processing (NLP) techniques and a trained machine learning model to classify email content.

## 🚀 Features

- Detects phishing/fake emails
- User-friendly web interface
- Machine Learning-based prediction
- Fast and accurate classification
- Simple and responsive design

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Sandhoshi08/Fake-Email-Detector.git
```

2. Move into the project

```bash
cd Fake-Email-Detector
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```
